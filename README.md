# Scrapping_alert
 Script simple en python pour Scrapper le prix d'un article sur Alibaba et renvoyer un alerte sur notre téléphone si le prix est en baisse avec un suivi de 24/7
# Installation
  # 1. Créer un environnement virtuel:
      python -m venv your_virtual_environment
  # 2. Intaller les dépendances:
      pip install -r requirements.txt
  # 3. Exécution
      python price_alert.py
